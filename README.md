# IM3
IM3 Projekt

**Kurzbeschreibung** (ca. 500 Zeichen)
In meinem Projekt soll aufgezeigt werden, wie viele Publibikes durchschnittlich in den Publibike Station in Bern zur Verfügung stehen. Diese Zahl vergleiche ich mit der Sonnenscheindauer sowie der Tageslichtzeit. Dazu habe ich die OpenMeteo API und die Citybikes API verwendet.

Dazu wurden alle vorhanden Publibike Zahlen im Raum Bern zusammengenommen und die Meteo Daten in Bern verwendet. Damit lässt sich herauslesen, wie viele Bikes durchschnittlich bei sonnigen - oder eben sonnenlosen - Tagen zur Verfügung stehen.
